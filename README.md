# vim-youcompleteme
vim-youcomplete me plugin for Ubuntu 16.04 etc. 

Usage:
1.  Download or clone this repository. 
2.  cd vim-youcompleteme-plugin/
3.  Copy/Move folder "vim" to your Home directory (/home/user).
4.  Rename "vim" to ".vim" ( note: there is a dot before vim).
5.  Do the same with vimrc file. 

So at the end, you home directory (/home/user) must have a folder named ".vim" and a file named ".vimrc"
